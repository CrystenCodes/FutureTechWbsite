Welcome to the Basics

Week 1: Introduction to ICT

Description: This lesson introduces the evolution of the web and modern ICT technologies.

The current state of ICT technologies explores how the internet has evolved over time.

Web 1.0 – The Read-Only Web

Description: The first generation of the internet focused on static information display.

Web 1.0 began in the mid 1990s and lasted until 2005. It was static, meaning users could only read content and could not interact or modify it.

Example of Web 1.0

Description: Early websites were simple and text-based.

One example of Web 1.0 was early Yahoo.com. The design was simple and content was static.

Web 2.0 – The Read-Write Web

Description: The modern interactive web where users create and share content.

Web 2.0 started in the early 2000s and continues today. It allows users to create, edit, and interact with content dynamically.

Web 3.0 – The Smart Web

Description: The future web focused on intelligence, automation, and security.

Web 3.0 includes advanced technologies, stronger cybersecurity, and smarter content systems.
