this lesson will teach you about web 2.0 and 3.0 but before those what came before? What is web 1.0? web 1.0 also known as web1, is the first generation web: this is where it began. It was made in the mid 1990's and it lasted until 2005

*web 1.0 is a static web, meaning that the content is fixed and cannot be changed by the user. *web 1.0 is also known as the "read-only" web, because users could only consume content, but they couldn't create or interact with it.

one of the few examples of web 1.0 is from yahoo.com


as you can see, the design is very simple and the content is static

web 2.0 is the second generation web: this is where we are now. It was made in the early 2000's and it is still evolving

web 2.0 is a dynamic web, meaning that the content can be changed by the user. web 2.0 is also known as the "read-write" web, because users can create and interact with content.
