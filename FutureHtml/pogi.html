Welcome to the Basics

Software:

Software is a set of instructions, programs, or data that tells a computer how to perform a task. Software differs from hardware, the physical part of a computer.

Software enables the computer to run under instructions for performing functions such as browsing the internet, editing documents, and playing games.

1. System Software:

The software controls hardware, manages hardware, and provides a platform for other software. Example: Operating Systems, Device Drivers, Utility Programs.

2. Application Software:

Programs designed to help users perform specific tasks. Example: Microsoft Word, Google Chrome, Games.
