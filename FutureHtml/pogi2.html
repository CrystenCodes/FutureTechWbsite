Rebooting & Formatting a PC

Rebooting: Restarting a computer, or rebooting, shuts down the operating system and immediately restarts it. This clears temporary memory (RAM), stops all running programs, and helps resolve minor glitches. Rebooting is the first step in many troubleshooting processes.

Formatting a PC: Formatting refers to erasing all data on a storage drive and preparing it for use. This is usually done before reinstalling an operating system or when clearing viruses, malware, or corrupted files. Formatting can be done on the entire system drive or on specific partitions.

Steps to Format a PC:

  1. Backup important files to an external storage device or cloud.
  2. Use a bootable USB or DVD with the operating system installer.
  3. Restart the PC and boot from the installation media.
  4. During installation, select the drive or partition to format.
  5. Confirm the formatting process. All data on that drive will be erased.
  6. Continue installation to set up a fresh operating system.

Important Notes: Formatting deletes all files permanently. Always backup your important data. Rebooting alone does not erase files—it just refreshes the system. Formatting is a deeper process and should only be done when necessary.
